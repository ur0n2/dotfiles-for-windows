Creator:  Jeff Pollock
Website:  LifeinPowerShell.blogspot.com
Download: https://onedrive.live.com/redir?resid=DF531FE80877C24F%21306

Description: 
A collection of PowerShell ISE color themes.

Instructions:
1. Extract the zipped files.
2. Open the PowerShell ISE.
3. Select Tools > Options.
4. Select Manage Themes.
5. Select Import and choose the theme you wish to import.
6. Now choose the theme in the list and apply it.
